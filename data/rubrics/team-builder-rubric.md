# Team Builder Rubric

Score each response on:

- `Build clarity`
  - states the real team goal cleanly
- `Team coherence`
  - the six slots belong to one shared game plan
- `Requested mon honesty`
  - does not silently force a weak request without saying so
- `Meta positioning`
  - build reflects the current Champions field
- `Role map quality`
  - each slot has a concrete job
- `Set direction practicality`
  - each slot gets usable lightweight direction, not just a label
- `Freshness and format handling`
  - states whether the build is a current-field recommendation or an inference-heavy early read when that matters
- `Practical usefulness`
  - the draft is something a competitive player could actually test

Failure triggers:

- generic six-goodstuff with no identity
- multiple half-committed versions instead of one recommendation
- no meaningful `Set Direction` guidance for the six slots
- claims current-field confidence without showing freshness or format grounding when the build depends on it
- matchup notes missing or detached from the actual team
