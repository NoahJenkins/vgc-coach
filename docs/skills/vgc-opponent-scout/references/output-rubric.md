# Output Rubric

Score each response on:

- `Evidence discipline`
  - separates known public info from extrapolation
- `Likely-shell quality`
  - identifies the most plausible opponent structures
- `Tendency usefulness`
  - highlights the few techs or habits that matter most
- `Prep value`
  - turns the scout into concrete preview or gameplan adjustments
- `Confidence honesty`
  - labels low-confidence reads clearly

Failure triggers:

- invented exact team information
- long lists of generic possibilities
- no confidence framing
- prep advice disconnected from the scout
