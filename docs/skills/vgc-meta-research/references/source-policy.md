# Source Policy

## Source Priority

Use this order:

1. official regulation/rules sources
2. official Play! Pokemon scheduling or format-transition sources
3. trusted community meta trackers and set databases
4. editorial/community commentary

## Rules

- Official sources define legality, mechanics, and active format.
- Community sources describe what players are doing inside that format.
- Never let a community source override an official rules source.

## Freshness Rules

- For current-meta questions, browse live first.
- Use absolute dates when clarifying currentness.
- If a source is early, sparse, or obviously volatile, call it early data.

## Conflict Handling

- If official and community sources conflict on legality, trust official.
- If community sources conflict on usage or team prevalence, report the disagreement and frame the conclusion directionally.
- Do not fabricate certainty by collapsing conflicting numbers into one claim.

## Drift Traps To Avoid

- stale Scarlet/Violet VGC assumptions
- mentioning inactive mechanics as live
- treating old usage articles as current format proof
- using one small sample as if it defines the whole field
