# Current Source Map

Use these sources in this order when researching the live Pokemon Champions meta.

## Official Format Sources

### Regulation Set M-A

Use for:

- active regulation dates
- active mechanics
- held item rules
- timers

Primary authority for legality and mechanics.

### Play! Pokemon transition announcement

Use for:

- when official VGC events move to Champions
- how Regulation I and Regulation M-A overlap during transition
- timing context for "current official play"

## Community Meta Sources

### ChampionsMeta

Use for:

- tournament-weighted usage leaders
- top tournament teams
- quick current competitive snapshot

Best when the question is about what is winning or appearing often in tracked tournament teams.

### Champions Lab

Use for:

- ladder plus tournament usage trends
- usage percentages with win rate context
- winning teams
- core pairs

Best when the question needs a broader "what real players are using right now" lens.

## Supporting Sources

### Pikalytics Champions

Use for:

- set tendencies
- common partners
- move, item, and ability expectations

Do not use it as the legality source of truth.

### Victory Road

Use for:

- event context
- tournament coverage
- human-written competitive interpretation

Use after the format and current usage picture are already grounded.

## Recommended Minimum Live Stack

For most current-meta questions, use at least:

1. official regulation source
2. one tournament-focused community source
3. one broader usage/community source

If those disagree materially, say so.
