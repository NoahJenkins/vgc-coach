# Format Checklist

## Lock The Check First

Before looking anything up, answer:

- What exact format or legality claim is being tested?
- Does the user need active-window confirmation, mechanic confirmation, roster legality, or all three?
- Is the real blocker uncertainty about the current regulation rather than the downstream coaching question?

## Verification Order

Use this order:

1. official regulation or event-policy source
2. official transition or announcement source
3. dated repo snapshot for support or fallback
4. community source only for context, not primary legality authority

## Good Habits

- Use absolute dates when clarifying what "current" means.
- Distinguish "confirmed active regulation" from "best current read."
- State whether the result is strong enough to support team-building or matchup advice immediately.

## Avoid These Failure Modes

- quoting community sites as if they define legality
- skipping the active date window
- hiding a rules gap behind confident prose
- answering a team question when the format basis is still uncertain
