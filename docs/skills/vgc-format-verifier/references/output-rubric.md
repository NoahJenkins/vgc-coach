# Output Rubric

Score each response on:

- `Check clarity`
  - states exactly what is being verified
- `Format confirmation`
  - identifies the active regulation or rules state cleanly
- `Legality precision`
  - separates what is legal, illegal, or still unclear
- `Inference honesty`
  - marks unsupported assumptions explicitly
- `Source quality`
  - prioritizes official sources and dates them appropriately
- `Decision usefulness`
  - makes clear whether downstream coaching can safely proceed

Failure triggers:

- assumes the format from memory
- uses community sources as primary legality proof
- collapses confirmed facts and inference together
- skips the confidence or gap statement
