# Journaling Checklist

## Start With The Session Goal

Before summarizing the games, answer:

- What was this session trying to improve?
- Was the focus matchup prep, lead discipline, preserve logic, or spread testing?

## Pattern Extraction

Look for:

- repeated lead or preview issues
- repeated preserve or sack errors
- recurring benchmark surprises
- repeated overperformance that should be preserved

## Good Journal Habits

- keep the recap short
- make `Keep` as concrete as `Change Next Session`
- turn the next session into a small set of testable adjustments

## Avoid These Failure Modes

- emotional recap with no pattern extraction
- one-game tilt turned into a broad conclusion
- no concrete next-session experiment
- only negatives, no record of what is actually working
