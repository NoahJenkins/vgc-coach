# Output Rubric

Score each response on:

- `Goal clarity`
  - states what the session was meant to improve
- `Compression quality`
  - summarizes the session without bloated recap
- `Pattern detection`
  - finds the real repeated themes
- `Balance`
  - preserves what worked and names what should change
- `Actionability`
  - gives concrete next-session adjustments and follow-up questions

Failure triggers:

- diary-like recap with no pattern extraction
- only negative findings
- vague next steps
- no separation between repeated issues and one-off events
