# Output Rubric

Score each response on:

- `Claim precision`
  - isolates the exact sentence being audited
- `Verdict quality`
  - picks the right support-strength label
- `Evidence separation`
  - distinguishes direct support from interpretation
- `Freshness handling`
  - dates the source and penalizes stale evidence appropriately
- `Trust framing`
  - explains why the source is strong, weak, partial, or misapplied
- `Writer usefulness`
  - leaves the next answer writer with a safer claim boundary

Failure triggers:

- no clear claim under review
- sources quoted without support analysis
- stale evidence presented as current proof
- unsupported claim treated as confirmed
