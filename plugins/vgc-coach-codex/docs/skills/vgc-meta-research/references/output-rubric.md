# Output Rubric

A strong response:

- confirms format, mechanics, and relevant date first
- identifies the real top threats and cores in the active format
- separates sourced claims from inference
- gives anti-meta openings that map to real team-building or prep decisions
- cites sources clearly

A weak response:

- opens with unsupported meta claims
- mixes formats
- hides uncertainty
- gives generic anti-meta advice that does not target specific teams or strategies
