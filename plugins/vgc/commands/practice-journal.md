# VGC Practice Journal

Use `$vgc-practice-journal` immediately for this request.

If "$ARGUMENTS" is non-empty, treat it as the user's session recap and start there.
If "$ARGUMENTS" is empty, ask only for the missing practice notes, session goal, or repeated review findings needed to proceed.

Preserve the skill's contract:

- compress the session into patterns, not a blow-by-blow log
- separate what should be kept from what should change next session
- make next-session changes concrete enough to test
- avoid vague "practice more" summaries

Recommended start:

`Use $vgc-practice-journal to turn this Pokemon Champions testing session into a concise practice journal with concrete next-session changes. $ARGUMENTS`
