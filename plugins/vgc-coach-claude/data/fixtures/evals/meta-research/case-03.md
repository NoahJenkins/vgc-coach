# Case 03

Request:

`How good is Sableye in the current Champions format? Be honest if the data is thin.`

Checks:

- does not invent strong usage support if data is thin
- gives a competitive-direct answer
- distinguishes thin-data inference from strong meta proof
- explains what role the mon can realistically play
- gives a realistic `use / do not use` framing rather than hedging forever
- stays focused on current fit in Champions, not old VGC history

Failure triggers:

- overstates Sableye as established top-meta if the live data does not support that
- hides uncertainty instead of labeling it
- gives only abstract role talk with no realistic shell or matchup context
