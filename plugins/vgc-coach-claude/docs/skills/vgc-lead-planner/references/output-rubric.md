# Output Rubric

A strong lead-planner answer:

- states a real default lead
- changes lead logic when the matchup actually changes incentives
- names preserve targets explicitly
- gives a usable turn-one objective for each matchup
- admits uncertainty where scouting gaps remain

A weak answer:

- repeats one lead pair for every matchup
- lists preserve targets without explaining why they matter
- gives generic advice like "play aggressively" or "position well"
- hides uncertainty behind fake confidence
