# VGC Meta Research

## Purpose

Use this skill to answer:

- what is popular right now
- what teams and cores are common
- what strategies are driving the format
- where anti-meta openings exist

## Inputs

- format/regulation
- optional archetype or target mon
- optional event lens

## Output Contract

Always return:

1. current format confirmation
2. top meta summary
3. common cores/teams
4. strategic trends
5. anti-meta openings
6. source list

## Rules

- official rules outrank community sites
- separate facts from inference
- if freshness is uncertain, say so
- do not use stale SV-era assumptions for Champions formats
