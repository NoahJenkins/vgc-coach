# VGC Team Audit

## Purpose

Use this skill to review a team and identify:

- structural problems
- matchup holes
- speed-control issues
- role overlap
- set tension

## Output Contract

Always return:

1. biggest issues first
2. what the team is trying to be
3. specific fixes
4. residual risk after fixes

## Rules

- findings first
- do not give vague "could use more synergy" feedback
- suggest concrete changes tied to actual problems
